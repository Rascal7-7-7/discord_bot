# Bot 機能・使い方まとめ

---

## 🗂️ Kura（Bot A）— 知識収集 & 検索Bot

**役割:** サーバー内の情報を蓄積し、いつでも引き出せるようにする

### スラッシュコマンド

| コマンド | 説明 | オプション |
|----------|------|------------|
| /search  | キーワードでメッセージを全文検索 | query（必須）, limit（1〜25, デフォルト10） |
| /recent  | 最近のメッセージを一覧表示 | channel（省略可 = 全チャンネル）, limit（1〜25, デフォルト10） |
| /summary | チャンネルの内容を統計要約 | channel（必須）, limit（5〜100, デフォルト30） |
| /help    | コマンド一覧表示 | なし |

### 自動反応機能

| トリガー | 動作 |
|----------|------|
| @Kura [質問] | AIメンターとして回答。スレッドを自動生成してその中に返信 |
| 追跡チャンネルにURLを投稿 | 記事・YouTube動画を自動取得してAIが要約。スレッドに投稿 |

### 追跡チャンネル（14チャンネル）

| カテゴリ | チャンネル |
|----------|------------|
| トレンド&リサーチ | トレンド速報, 競合・市場分析, ひらめきアイデア |
| スキル保管庫 | 動画編集-tips, デザイン-tips, aiプロンプト集, windows-tips, mac-tips, マーケ・営業術, マネーリテラシー, 入門教材リンク |
| AIアシスタント・効率化 | aiメンター相談, 要約・ドラフト生成, 自動化ログ |

### /summary の出力内容

- 期間・メッセージ数
- アクティブユーザーTop5（投稿数付き）
- 頻出ワードTop10

## 📡 Pulse（Bot B）— トレンド自動収集Bot

**役割:** AI・テック・マーケティングの最新トレンドを毎日自動でレポート

### 自動実行スケジュール

| タイミング | 動作 |
|------------|------|
| 毎日 9:00 JST  | トレンドレポートを自動投稿 |
| 毎日 21:00 JST | トレンドレポートを自動投稿 |
| 起動時         | 即座に1回実行 |

### データソース（現在4系統）

| ソース | 内容 |
|--------|------|
| Hacker News | エンジニア・スタートアップ界隈のホット記事（上位15件） |
| Dev.to | 開発者向け技術記事（トレンドTop10） |
| Lobste.rs | プログラミング特化コミュニティのホット記事 |
| GitHub Trending | 直近7日で急上昇中のAI/LLMリポジトリ |

### 投稿フォーマット

1. #トレンド速報 にサマリーメッセージを投稿  
2. そこからスレッドを自動作成し、各アイテムをEmbedカードで投稿  

### 各カードの内容

- タイトル・URL・ソース
- 対象ペルソナ（動画編集者 / SNS運用者 / マーケター / デザイナー / ビジネス全般）
- 重要な理由（40字以内）
- 活用法（40字以内）
- 絵文字アイコン

### ログ

- #自動化ログ に実行結果（件数・エラー）を自動記録
