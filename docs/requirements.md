# 要件定義書

## プロジェクト概要

Discord サーバー内の技術情報・共有情報を蓄積し、検索・要約・再利用できる Bot。

## 目的

- チャンネルに流れる情報の消失を防ぐ
- 過去の議論・ナレッジを検索可能にする
- チーム内の情報共有を効率化する

## 想定ユーザー

- 技術チームのメンバー（エンジニア、PM）
- 情報を後から参照したいすべてのサーバーメンバー

## MVP 機能一覧

| コマンド | 説明 |
|----------|------|
| `/search` | キーワードで過去メッセージをAND検索 |
| `/recent` | 最近のメッセージ一覧を表示 |
| `/summary` | チャンネルの簡易要約（投稿者・頻出ワード） |
| `/help` | Bot の使い方を表示 |

## 非機能要件

- PostgreSQL（Neon）でデータ永続化
- Google Cloud Run でホスティング
- Bot / 分析サービスの分離（モノレポ）

## 対象外（MVP）

- 意味検索・AI 埋め込み
- Web ダッシュボード
- 複数サーバー同時対応
